title: "Fine-tuning with Azure OpenAI"
description: "Fine-tune your models with Azure OpenAI"
---

Azure OpenAI follows the same fine-tuning process as OpenAI. Please refer to the [OpenAI fine-tuning documentation](/integrations/llms/openai/fine-tuning) for more details.

Azure OpenAI fine-tuning API is available at [Azure OpenAI API](https://learn.microsoft.com/en-us/rest/api/azureopenai/fine-tuning/create?view=rest-azureopenai-2025-01-01-preview&tabs=HTTP).
