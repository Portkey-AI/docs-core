[Xcode](https://developer.apple.com/xcode/) is Apple's integrated development environment with AI-powered coding assistance. Add Portkey as your model provider to get:

- **1600+ LLMs** — Use any provider (OpenAI, Anthropic, Google, etc.) through one gateway
- **Observability** — Track costs, tokens, and latency for every request
- **Governance** — Budget limits, usage tracking, and team access controls
- **Reliability** — Automatic fallbacks, retries, and caching

This guide shows how to configure Xcode to use Portkey as an OpenAI-compatible endpoint.

<Warning>
**Known Limitations**
- Xcode uses the **Chat Completions API route** by default
- **GPT-5.3 Codex only supports the Responses API route**, making it incompatible with Xcode
- Use models that support Chat Completions (e.g., GPT-4o, GPT-4.1, Claude, etc.) for full compatibility
</Warning>

<Note>
For enterprise deployments across teams, see [Enterprise Governance](#3-enterprise-governance).
</Note>

## 1. Setup

<Steps>
<Step title="Add Provider">
Go to [Model Catalog](https://app.portkey.ai/model-catalog) → **Add Provider**.

<Frame>
<img src="/Screenshot2025-07-21at5.29.57PM.png" width="500"/>
</Frame>
</Step>

<Step title="Configure Credentials">
Select your provider (OpenAI, Anthropic, etc.), enter your API key, and create a slug like `openai-prod`.

<Frame>
<img src="/images/product/model-catalog/create-provider-page.png" width="500"/>
</Frame>
</Step>

<Step title="Create Config">
Go to [Configs](https://app.portkey.ai/configs) and create a config to route requests:

```json
{
  "override_params": {
    "model": "@openai-prod/gpt-4o"
  }
}
```

Save and note the Config ID.

<Note>
**Important:** Choose a model that supports Chat Completions. GPT-5.3 Codex is not supported as it only works with the Responses API.
</Note>
</Step>

<Step title="Get Portkey API Key">
Go to [API Keys](https://app.portkey.ai/api-keys) → Create new key → Attach your config → Save.
</Step>
</Steps>

## 2. Configure Xcode

<Steps>
<Step title="Open Xcode Settings">
In Xcode:
1. Open **Settings** (macOS: `Cmd + ,`)
2. Navigate to the **Intelligence** tab
</Step>

<Step title="Configure ChatGPT Model">
1. Under **Predictive Code Completion Model**, select **ChatGPT**
2. Click **Set Up** or **Change** next to ChatGPT
3. Select **Use a different OpenAI-compatible LLM server**
4. Enter the following details:
   - **Server URL:** `https://api.portkey.ai/v1`
   - **API Key:** Your Portkey API key from step 1
5. Click **Done** to save
</Step>
</Steps>

Done! Xcode's AI features will now route through Portkey. Monitor usage in the [Portkey Dashboard](https://app.portkey.ai/dashboard).

## Supported models

Since Xcode uses Chat Completions, use models that support this API:

| Provider | Recommended Models |
|----------|-------------------|
| OpenAI | gpt-4o, gpt-4.1, gpt-4-turbo |
| Anthropic | claude-3-5-sonnet, claude-3-opus |
| Google | gemini-1.5-pro, gemini-2.0-flash |
| Azure OpenAI | gpt-4o, gpt-4 |

<Warning>
**Not Supported:** GPT-5.3 Codex (Responses API only)
</Warning>
