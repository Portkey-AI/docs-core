---
title: "Delete SCIM Workspace Mapping"
openapi: delete /scim/workspaces/{scimWorkspaceGroupId}
---

<Info>
Deleting a mapping affects only the workspace it points to. If the same SCIM group is mapped to additional workspaces, those mappings, and the user memberships they provisioned, remain intact.

Users who were provisioned into the targeted workspace only through this mapping are removed from that workspace. The organization's default/shared workspace is never archived as part of the cleanup; non-default workspaces may be archived if no users remain after the mapping is removed.
</Info>
