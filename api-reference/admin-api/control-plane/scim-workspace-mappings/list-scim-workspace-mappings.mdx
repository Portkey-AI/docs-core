---
title: "List SCIM Workspace Mappings"
openapi: get /scim/workspaces
---

<Info>
The response is paginated. The query parameters `page` (zero-indexed) and `page_size` control pagination, and the response includes a `total_count` field reflecting the total number of mappings across all pages.

When a SCIM group is mapped to multiple workspaces, the list contains one entry per `(group, workspace)` pair. Filter by `workspaceId`, `scimGroupId`, or `role` to narrow results.
</Info>
