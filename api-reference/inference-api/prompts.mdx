---
title: "Overview"
description: Prompts in Portkey allow you to create and manage prompt templates collaboratively.
---

The `portkey.prompts` APIs allow you to

* [Trigger a prompt completion](/portkey-endpoints/prompts/prompt-completion)
* [Retrieve a saved prompt](/portkey-endpoints/prompts/render)

While, the following methods are in private beta

* List all Prompts
* Get a Prompt
* Create a new Prompt
* Update an existing Prompt
* Delete a Prompt
